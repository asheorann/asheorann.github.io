# Project Images

Add your project screenshots, diagrams, and visuals here.

## How to Use

1. **Add your image** to this folder (e.g., `safety-research.png`)

2. **Update your project frontmatter** in `src/content/projects/your-project.mdx`:
   ```yaml
   ---
   title: "Your Project Title"
   startDate: 2026-01-15
   summary: "Project description..."
   tags: ["research", "technical"]
   featured: true
   coverImage: "/images/projects/safety-research.png"
   ---
   ```

3. **The image will appear** at the top of the project card with a subtle zoom effect on hover

## Image Guidelines

- **Format:** JPG or PNG (PNG preferred for diagrams/screenshots)
- **Size:** 800x450px or 16:9 aspect ratio recommended
- **File size:** Keep under 300KB for fast loading (use tools like TinyPNG or Squoosh)
- **Content ideas:**
  - Architecture diagrams
  - Research graphs/visualizations
  - Dashboard screenshots
  - Project interface mockups
  - Results/findings visualizations

## Examples

Good filenames:
- `clariem-dashboard.png`
- `safety-research-results.jpg`
- `pharma-forecasting-ui.png`
- `claude-club-event.jpg`
