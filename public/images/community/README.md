# Community Images

Add your community photo here as `community-photo.jpg`.

## Guidelines

- Name the file exactly: `community-photo.jpg` (or `.png`)
- Recommended size: 400-800px wide for landscape, or 300px tall for portrait
- The image will be displayed in the community section on the homepage
- Update the `alt` attribute in `src/pages/index.astro` to describe the photo
- Update the caption text in the same file (currently `[caption goes here]`)
