# Images Folder

## profile/
Add your profile photo here:
- `profile-photo.jpg` or `profile-photo.png` - Your headshot/profile picture

## logos/
Add your university and organization logos here:
- `ucsd-logo.png` - UCSD logo (timeline)
- `penn-logo.png` - Penn logo (timeline)
- `highschool-logo.png` - Your high school logo or Canada flag (timeline)
- `clariem-logo.png` - Clariem logo (optional - for projects)
- `curious-cardinals-logo.png` - Curious Cardinals logo (optional - for projects)

## projects/
Add project screenshots, thumbnails, or visuals here:
- Name files descriptively (e.g., `clariem-dashboard.png`, `safety-research-diagram.jpg`)
- Reference them in your project frontmatter as: `coverImage: "/images/projects/your-file.png"`
- Images will appear at the top of project cards with a subtle zoom effect on hover

## Recommended sizes:
- Timeline logos: 32x32px or 40x40px (small, will be displayed in circles)
- Profile photo: 400x400px or larger (square, will be displayed as circle)
- Project thumbnails: 800x600px or 16:9 ratio

## Formats:
- PNG with transparency preferred for logos
- JPG or PNG for photos
- SVG for simple logos (scalable and small file size)
- Keep file sizes under 500KB for performance

## Tips:
- Use high-quality images (at least 2x the display size for retina screens)
- Logos should have transparent backgrounds
- Profile photo should be well-lit and professional
- Compress images before uploading (use tools like TinyPNG or Squoosh)
